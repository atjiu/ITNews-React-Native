# ITNews

IT资讯APP，使用React Native编写

## 开始

- clone 本项目
- 安装package `yarn install`
- 运行在模拟器里 `react-native run-ios` or `react-native run-android` （其中run-android要保证电脑上启动了一个Android模拟器，或者手机连接到电脑上了）
